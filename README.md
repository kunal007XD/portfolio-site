# Kunal Mehta — React + Tailwind Profile

This repository contains a React + Vite single-page profile site built with TailwindCSS, Framer Motion and Recharts.

## Quickstart

1. Install dependencies
```bash
npm install
```

2. Start the dev server
```bash
npm run dev
```

3. Build for production
```bash
npm run build
npm run preview
```

## Notes

- Tailwind CSS is configured in `tailwind.config.cjs`
- Put a `resume.pdf` file in the `public/` folder to enable the "Download PDF" button.
- Replace contact links in `src/App.jsx` as needed.
